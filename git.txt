Commands
  Local:
  - git init to initialize a local git repository
  - git add <file> add to staging area
  - git rm --cached <file> remove from staging area
  - git status to check files in staging area
  - git commit to commit changes to local repository
      -m 'comment' to add a comment
      -in vim type :wq to save
  - touch <file> to create a new empty file
  - .gitignore a file to add file names to that you dont want to include
  - branches:
        git branch <branch name> to create a new brach
  - git merge <branch name>

  Remote: Github
  - git push push to remote repository
  - git pull pull latest changes from repository
  - git clone <file thingy> to copy a repository onto your machine
